# Baccarat Simulator

A command-line Baccarat casino game simulator built in C++.

## Features
- Full baccarat rules implementation including third-card drawing
- Banking system with balance tracking
- Multiple betting options (Player, Banker, Tie)
- Persistent player data
- Game statistics and history
